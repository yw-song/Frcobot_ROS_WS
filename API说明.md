## 机器人运动 API
不同的机器人运动有着不同的含义，分别有：

1. **点动（Jog）**：用于机器人在关节或笛卡尔空间中进行小幅度的手动调整。
   - 例子：沿X轴小步移动机械臂。

2. **笛卡尔空间运动**：通过XYZ坐标和方位控制机器人运动。
   - 例子：将机械臂末端移动到工作空间的特定位置。

3. **伺服运动**：实现对机器人关节的实时、精确控制，使运动平滑连续。
   - 例子：让机械臂沿设定路径平稳运动。

4. **样条运动**：使用样条插值在复杂路径上实现平滑运动。
   - 例子：机械臂平滑经过多个路径点。

可以参考[文档](https://fr-documentation.readthedocs.io/zh-cn/latest/SDKManual/CPPRobotMovement.html)获取更多详细信息。
